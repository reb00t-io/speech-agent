## About this app

This is a single chat assistant with text and voice.

- **Text chat**: type a message and press Enter (Shift+Enter for a newline).
- **Voice conversation**: the round voice button starts a live, hands-free conversation — speak and the assistant replies out loud. Start speaking again to interrupt it.
- **Dictation**: the microphone button transcribes what you say into the text box so you can edit and send it.
- **Image upload**: attach an image with the "+" button to ask about it.
- **Web search**: enable the web-search shortcut to have the assistant look things up online before answering.
- **Deep research**: enable the deep-research shortcut to have the assistant investigate a topic thoroughly and deliver a downloadable PDF report.
